Data Visualisation

EffectiveAltruismData.com

Effective Altruism (EA) is a collective of quantitatively-minded philanthropists and do-gooders. There are many EA organisations who collect data on grants, donors, pledges, and so on, so I built this website to aggregate and visualise this data.

The website is coded in Python using Dash and Plotly, and deployed with Heroku. And it's responsive!
